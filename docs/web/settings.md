# Settings Page

Access at `http://Car-HUD.local:8080/settings`

## Sections

### WiFi
- Scan for available networks
- Connect with SSID and password
- Shows current connection status

### Bluetooth
- Scan for nearby devices (8 second scan)
- Pair and connect to phones
- Toggle Spotify Connect (Raspotify)

### Theme
- Select from: Auto, Blue, Red, Green, Amber, Day, Night
- Auto switches between Day/Night based on time

### Display
- Brightness slider (1-100%)
- Real-time adjustment via PWM

### Widgets
- Toggle each widget ON/OFF
- Changes take effect immediately

### System
- OBD connection status
- Dashcam recording status and storage
- Microphone status

### Now Playing
- Current track and artist
- Playback device
